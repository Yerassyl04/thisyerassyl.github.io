A real-time face authentication system built in Python. The core contribution is an
anti-spoofing (liveness detection) module that distinguishes a live subject from
presentation attacks such as printed photos and video replays, strengthening the
security of the authentication pipeline.

**Highlights**

- Implemented an anti-spoofing algorithm to separate live faces from photo/video replays.
- Built a real-time recognition pipeline for authentication scenarios.
- Focused on practical security hardening of biometric access.

**Stack:** Python, OpenCV, deep learning.
